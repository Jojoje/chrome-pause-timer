# Privacy Statement Summary

Mindful Site Timer uses only the minimum data needed to provide its core feature: time limits on user-selected websites.

It stores rule settings and usage totals locally in your browser extension storage.

Data is not sent to external servers, not sold, not shared for advertising, and not used for analytics.

Host access is requested only when you add a site rule.

If you uninstall the extension, Chrome removes extension data from your browser profile.
