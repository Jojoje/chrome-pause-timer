# Full Description

Mindful Site Timer is a lightweight Chrome extension for reducing doom-scrolling and helping you browse with intention.

## What It Does

- Set daily limits for specific sites
- Target entire sites or specific paths, like YouTube Shorts only
- Optionally include subdomains per rule
- Use immediate block mode by leaving daily minutes empty
- Show a short mindful pause screen when time is up
- Add small same-day extensions (1, 2, 5, 10 minutes) when needed
- Reset usage automatically at local midnight or your custom reset time

## How It Works

- You add the sites you want to manage in Settings
- The extension tracks active-tab time on those sites
- When your daily allowance is reached, the page is replaced with a short breathing-style pause experience
- After the pause, you can either stop browsing or extend for a few minutes

## Privacy-First by Design

- No ads
- No analytics
- No remote servers
- No sale or sharing of data
- All settings and usage data are stored locally in your browser

Mindful Site Timer is built to stay focused, simple, and transparent.
