# What's New (1.0.1)

Mindful Site Timer 1.0.1 improves real-world browsing workflows and pause consistency.

- Track time across multiple visible windows (not just the currently focused window)
- Pause countdown now requires tab focus and resets on focus loss
- Extending time in one tab now dismisses matching pause overlays in other tabs
- Slower breathing animation for a calmer rhythm
- Cleaner site list UX with compact cards and inline edit support
