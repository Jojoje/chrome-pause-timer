# Privacy Policy for Mindful Site Timer

Last updated: 2026-07-01

## Overview

Mindful Site Timer is a Chrome extension designed to help users limit time spent on distracting websites.

This extension is privacy-first by design:

- No ads
- No analytics
- No remote backend
- No sale or sharing of user data

## Data We Process

Mindful Site Timer processes only data required for its functionality:

1. Site rules configured by the user
   - Example: tracked host, optional path, subdomain setting, daily time allowance
2. Usage totals
   - Time spent on tracked sites for the current and recent days
3. Extension settings
   - Example: daily reset time and pause duration

## Where Data Is Stored

All extension data is stored locally using Chrome extension local storage.

No browsing usage data is transmitted to any external server.

## How Data Is Used

Data is used only to provide core extension features:

- Applying your configured site limits
- Triggering the mindful pause overlay when limits are reached
- Allowing optional short extensions for the same day
- Resetting daily usage at your configured reset time

## Permissions

Mindful Site Timer requests only permissions needed for its operation:

- storage: save rules, settings, and usage locally
- alarms: periodic timing updates
- tabs: determine active tab for usage timing
- scripting: register content scripts on approved hosts
- optional host permissions: requested when the user adds a site rule

## Data Sharing and Selling

Mindful Site Timer does not:

- Sell user data
- Share user data with advertisers or data brokers
- Send user data to third-party analytics providers

## Security

The extension operates entirely on-device and does not rely on remote code execution.

## User Control

Users can:

- Add or remove site rules at any time
- Change limits and reset settings
- Uninstall the extension to remove stored extension data from the browser profile

## Changes to This Policy

If this policy changes, the updated version will be published with a new "Last updated" date.

## Contact
